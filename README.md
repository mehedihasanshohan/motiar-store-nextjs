# Motiar Store - Modern Grocery E-commerce

**মতিয়ার স্টোর** একটি আধুনিক ই-কমার্স ওয়েব অ্যাপ্লিকেশন যা বিশেষভাবে আমার বাবার মুদি দোকানের জন্য ডিজাইন এবং ডেভেলপ করা হয়েছে। এটি Next.js এবং Tailwind CSS ব্যবহার করে তৈরি করা একটি হাই-পারফরম্যান্স শপিং প্ল্যাটফর্ম।

---

## Short Project Description
এই প্রজেক্টটির মূল লক্ষ্য হলো একটি সাধারণ মুদি দোকানের ব্যবসায়িক কার্যক্রমকে ডিজিটাল রূপান্তর করা। কাস্টমাররা যাতে সহজে দোকানের পণ্যগুলো দেখতে পারে, দাম যাচাই করতে পারে এবং কেনাকাটা করতে পারে, সেই উদ্দেশ্যেই এটি তৈরি। এর প্রিমিয়াম ডিজাইন এবং দ্রুতগতির ইউজার ইন্টারফেস কেনাকাটার অভিজ্ঞতাকে সহজ ও আনন্দদায়ক করে তোলে।

---

## Implemented Features
- **Modern Responsive UI:** মোবাইল, ট্যাবলেট এবং ডেস্কটপ—সব ডিভাইসের জন্য অপ্টিমাইজড।
- **Dynamic Product Grid:** ক্যাটাগরি অনুযায়ী পণ্য প্রদর্শনের ব্যবস্থা।
- **Detailed Product View:** প্রতিটি পণ্যের জন্য আলাদা ডিটেইলস পেজ।
- **Weekly Mega Offers:** ডিসকাউন্ট আইটেমগুলোর জন্য বিশেষ আকর্ষণীয় সেকশন।
- **Shop Features Section:** কেন আমরা সেরা—সেটি হাইলাইট করার জন্য ট্রাস্ট বিল্ডিং সেকশন।
- **Fast Navigation:** Next.js App Router ব্যবহার করে দ্রুত পেজ ট্রানজিশন।

---

## Brief Explanation of Key Features
- **Server-Side Fetching:** ডাইনামিক ডাটা সরাসরি সার্ভার থেকে ফেচ করা হয়, যা এসইও (SEO) ফ্রেন্ডলি।
- **Clean Aesthetic:** পুরো সাইটে Cyan-700 এবং White কালার স্কিম ব্যবহার করা হয়েছে একটি ফ্রেশ এবং ফ্রেন্ডলি লুক দেওয়ার জন্য।

## Route Summary
| Route | Description |
| :--- | :--- |
| `/` | **Home Page:** ব্যানার, ধামাকা অফার এবং ফিচার পণ্যসমূহ। |
| `/products` | **All Products:** দোকানের সব পণ্যের তালিকা। |
| `/products/[id]` | **Product Details:** পণ্যের বিস্তারিত বিবরণ এবং স্পেসিফিকেশন। |
| `/about` | **About Us:** দোকানের ইতিহাস ও উদ্দেশ্য। |
| `/contact` | **Contact:** দোকানের অবস্থান ও যোগাযোগের তথ্য। |

## Setup & Installation Instructions

১. প্রজেক্টটি ক্লোন করুন:
```bash
